# wifi-security

> Get current wifi network security type.

# Install

```bash
npm install wifi-security
```

# Usage

```js
const wifiSecurity = require('wifi-security')

console.log(wifiSecurity()) // "wpa2-psk"
```

# CLI

```bash
npm install -g wifi-security
```

```bash
$ wifi-security
wpa2-psk
```
